# Math Operators
From highest to lowest precedence:
|Operators|Operation|Example|
|-|-|-|
|**|Exponent|2 ** 3 = 8|
|%|Modulus/Remainder|10 % 3 = 1|
|//|Integer division (Floor division)|10 // 3 = 3|
|/|Division|10 / 3 = 3.333335|
|*|Multiplication|10 * 3 = 30|
|-|Subtraction|10 - 3 = 7|
|+|Addition|10 + 3 = 13|
